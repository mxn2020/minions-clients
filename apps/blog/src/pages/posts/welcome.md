# Welcome to Minions Clients

We're excited to introduce **Minions Clients** — Client profiles built from job interactions, communications, and contracts.

## What's Included

- **TypeScript SDK**: `npm install @minions-clients/sdk`
- **Python SDK**: `pip install minions-clients`
- **CLI**: `npm install -g @minions-clients/cli`

## Getting Started

Check out the [documentation](https://clients.minions.help) for a full guide.
