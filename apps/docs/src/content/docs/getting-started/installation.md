---
title: Installation
description: Install Minions Clients packages
---

## TypeScript / Node.js

```bash
npm install @minions-clients/sdk minions-sdk
```

## Python

```bash
pip install minions-clients
```

## CLI

```bash
npm install -g @minions-clients/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
