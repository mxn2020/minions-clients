# Minions Clients

Client profiles built from job interactions, communications, and contracts

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-clients/sdk`) — Core library for Node.js
- **Python SDK** (`minions-clients`) — Python bindings
- **CLI Tool** (`@minions-clients/cli`) — Command-line interface
- **Web App** — Interactive playground at [clients.minions.wtf](https://clients.minions.wtf)
